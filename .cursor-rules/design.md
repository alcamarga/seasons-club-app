# 🎨 Estándares de Diseño

## 1. Identidad Visual
- **Paleta de Colores:** Basada en Pizzería (Rojos #FF0000, Naranjas, Blancos).
- **Tipografía:** Limpia y legible (Sans-serif).

## 2. Componentes UI (Bootstrap 5)
- **Tablas:** Siempre con la clase `table table-striped table-hover`.
- **Botones:** Usar clases semánticas (`btn-primary` para acciones principales, `btn-danger` para cancelar).
- **Responsividad:** Todo debe verse bien en móviles y tablets de cocina.

## 3. UX (Experiencia de Usuario)
- Los mensajes de error deben ser amigables.
- Siempre mostrar un "spinner" o mensaje de carga mientras se obtienen datos del API.
