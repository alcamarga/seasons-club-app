# 📋 Requerimientos del Proyecto

## 1. Visión General
- El sistema debe permitir la gestión de pedidos en tiempo real para una Pizzería.
- Debe ser ligero, rápido y fácil de usar en una cocina.

## 2. Funcionalidades Core (MVP)
- **Visualización:** Tabla de pedidos con estados (En Horno, Empacando, Entregado).
- **Conectividad:** El Frontend debe reflejar cambios realizados en el Backend.
- **Persistencia:** Los datos deben guardarse de forma segura.

- "Milo, si una funcionalidad que sugieres no está en este archivo o no ayuda directamente a estos puntos, consúltala con Camilo antes de codear."
